# Braph-2.0-Python README

## Installation

    pip3 install .

## Tests

### Unit Tests

To run all tests:

    python3 -m unittest

To run a single test file:

    python3 -m unittest test.test_betweenness

To debug a test in Visual Studio Code:

1. Set appropriate breakpoints
1. Press F5
1. Select Module
1. Type for example test.test_distance
